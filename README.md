# 🌐 Konnect Frontend

A modern, responsive frontend for **Konnect**, a full-stack social networking platform that enables users to connect, chat in real time, share media, and make peer-to-peer video calls.

Built with **React**, the application provides an intuitive user experience with secure authentication, real-time messaging, responsive UI components, and seamless integration with the Konnect backend.

---

# ✨ Features

* 🔐 Secure Authentication (JWT)
* 👤 User Profile Management
* 🤝 Send & Manage Connection Requests
* 💬 Real-time Chat using Socket.io
* 📷 Image Sharing
* 📞 WebRTC Video Calling
* 👥 Group Chats
* 🟢 Online User Status
* 🔍 User Discovery
* 📱 Fully Responsive Interface
* ⚡ Fast Navigation with React Router
* 🎨 Clean and Modern UI

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/Hemansh-sharma/Konnect--frontend.git
```

Move into the project

```bash
cd Konnect--frontend
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
VITE_API_URL=http://localhost:5000
```

Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

*(Update the port if your project uses a different development server.)*
